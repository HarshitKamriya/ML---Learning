## ğŸ”‘ What I Learned in ML Algorithms

### 1. Linear Regression
- **Concept:** Predict continuous values using a straight-line fit.
- **Key Learnings:**
  - Gradient Descent optimization
  - Overfitting vs. underfitting
- **Demo:** House price prediction

### 2. Logistic Regression
- **Concept:** Classification using sigmoid function.
- **Key Learnings:**
  - Binary classification
  - Decision boundaries
- **Demo:** Spam email detection

### 3. Decision Trees
- **Concept:** Splitting data into branches based on features.
- **Key Learnings:**
  - Gini impurity & entropy
  - Pruning to avoid overfitting
- **Demo:** Customer churn prediction

### 4. Random Forest
- **Concept:** Ensemble of decision trees.
- **Key Learnings:**
  - Bagging technique
  - Feature importance
- **Demo:** Loan approval prediction

### 5. Support Vector Machines (SVM)
- **Concept:** Find hyperplane that best separates classes.
- **Key Learnings:**
  - Kernel trick
  - Margin maximization
- **Demo:** Handwritten digit classification

### 6. K-Nearest Neighbors (KNN)
- **Concept:** Classify based on closest neighbors.
- **Key Learnings:**
  - Distance metrics (Euclidean, Manhattan)
  - Effect of K value
- **Demo:** Iris dataset classification

### 7. Neural Networks (Basics)
- **Concept:** Layers of interconnected nodes.
- **Key Learnings:**
  - Forward & backward propagation
  - Activation functions
- **Demo:** MNIST digit recognition
